url

统一资源定位符是对可以从互联网上得到的资源的位置和访问方法的一种简洁的表示，是互联网上标准资源的地址。互联网上的每个文件都有一个唯一的URL，它包含的信息指出文件的位置以及浏览器应该怎么处理它。

url编码

url编码是一种浏览器用来打包表单输入的格式。浏览器从表单中获取所有的name和其中的值 ，将它们以name/value参数编码（移去那些不能传送的字符，将数据排行等等）作为URL的一部分或者分离地发给服务器。

题目描述

找素数
Your task is simple:
Find the first two primes above 1 million, whose separate digit sums are also prime.
As example take 23, which is a prime whose digit sum, 5, is also prime.
The solution is the concatination of the two numbers,
Example: If the first number is 1,234,567
and the second is 8,765,432,
your solution is 12345678765432

写程序

C++

#include<iostream>
#include<math.h>
using namespace std;

int a;
//判断n为素数的函数，将其%2~sqrt(n)，应为其因子必然一个大于sqrt而另一个小于，所以不用算到n-1
bool IsPrime(int p) {        
a = (int)sqrt(p) ;
for (int i = 2; i <= a; i++) {
    if ((p%i )== 0)
        return false;
}
return true;
}
// 求各位数之和
int Add_Number(int number) {
int add_number = 0;
int one;
while (number != 0) {
     one= number % 10;
     add_number += one;
     number = number / 10;
}
return add_number;
}

int main() {
int add_number;
for (int number = 1000000;; number++) {//先取number为1000000，运行程序的结果+1后再设其为number再次运行程序
    add_number = Add_Number(number);
    if (IsPrime(number)&&IsPrime(add_number)) {
        cout << number << endl;
        return 0;
        }
    }
}
